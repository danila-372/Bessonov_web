# Лабораторная работа 4

Flask-приложение для управления учетными записями пользователей.

## Возможности

- аутентификация пользователей с загрузкой данных из SQLite;
- просмотр списка пользователей и карточки пользователя;
- создание, редактирование и удаление пользователей;
- серверная валидация логина, пароля и обязательных полей;
- смена пароля текущего пользователя;
- flash-сообщения об ошибках и успешных операциях;
- pytest-тесты для реализованных сценариев.

При первом запуске автоматически создаются роли и учетная запись администратора:

- логин: `admin`
- пароль: `Admin123!`

## Запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
flask --app run.py run
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:5000`.

## Тесты

```powershell
pytest
```
