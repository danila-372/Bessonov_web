# АИС «Электронная библиотека»

Учебное Flask-приложение для ведения каталога книг, поиска, рецензий и пользовательских подборок.

## Возможности

- каталог книг с поиском, сортировкой по году издания и пагинацией по 10 книг;
- роли `admin`, `moderator`, `user` и разграничение доступа;
- добавление, редактирование и каскадное удаление книг;
- загрузка обложек с дедупликацией файлов по MD5-хэшу;
- Markdown-описания и рецензии с очисткой потенциально опасного HTML через Bleach;
- одна рецензия пользователя на одну книгу;
- пользовательские подборки книг для роли `user`;
- удаление рецензий администратором или модератором.

## Запуск

```powershell
python -m pip install -r requirements.txt
python -m flask --app app init-db
python -m flask --app app run --debug
```

После запуска приложение доступно по адресу <http://127.0.0.1:5000>.

## Тестовые пользователи

| Роль | Логин | Пароль |
| --- | --- | --- |
| Администратор | `admin` | `admin` |
| Модератор | `moderator` | `moderator` |
| Пользователь | `user` | `user` |

Жанры, роли и тестовые пользователи создаются командой `init-db`. База данных находится в `instance/library.db`, загруженные обложки — в `instance/uploads`.
